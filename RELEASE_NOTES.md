# Release Notes

Version v1.40.3 — March 3, 2026

## Boon Uptime: subgroup entries and UI improvements
- Boon uptime in the stats view now supports subgroup entries, so uptime can be shown broken out by subcategories.
- UI elements in this panel are clearer and easier to interact with, making it faster to scan and compare uptime data.
- Existing data remains compatible; subgroup entries appear where applicable.
